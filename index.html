Palettr is a simple color palette creation tool. Palettr is all about easily creating color palettes and sharing them with others.

With Palettr you can create color palettes and share them with friends and clients. To share your color palettes just copy the link provided and share that on your favourite social networks.

If you have a color palette that is too large to share or if you just want to be able to save your color palette you can export it. When you click on the export button Palettr will ask if you want to download the file. You can share this file if you like. To view the color palette that you exported just click on the import button and select the exported color palette file. Once imported you should be able to see all of your color palettes that are part of the export file.

TIP: You can also import multiple color palette files and then export all the imported files as one big export :)
